## 一、产品/工具类词根

这类词根通常用于描述能够帮助用户完成特定任务的产品或工具。

### 1.1 Generator（生成器）

| 属性 | 描述 |
|------|------|
| **常见组合** | random number generator, ai image generator, qr code generator, font generator, citation generator |
| **用户意图** | 功能性/重复性内容生成 |
| **商业机会** | AI 生成器（针对特定领域的测试数据、特定艺术风格等） |

---

### 1.2 Creator（创作者）

| 属性 | 描述 |
|------|------|
| **常见组合** | bing image creator, logo creator, meme creator, floor plan creator |
| **用户意图** | 面向个人的创意活动（视频、写作、艺术） |
| **商业机会** | 自助创作工具；头脑风暴和创意辅助工具 |

---

### 1.3 Maker（制作器）

| 属性 | 描述 |
|------|------|
| **常见组合** | logo maker, gif maker, meme maker, coffee maker, song maker |
| **用户意图** | 制作特定成品的工具 |
| **商业机会** | 快速网站/落地页构建器；教育资源平台 |

---

### 1.4 Builder（构建器）

| 属性 | 描述 |
|------|------|
| **常见组合** | resume builder, website builder, schedule builder, career builder |
| **用户意图** | 从预设组件构建系统/产品 |
| **商业机会** | 自动化工具站；可视化 AI 工作流构建器 |

---

### 1.5 Constructor（构造器）

| 属性 | 描述 |
|------|------|
| **常见组合** | f1 constructor, java constructor |
| **用户意图** | 技术/编程相关的构造器；面向对象编程函数 |
| **商业机会** | AI 代码编程相关工具 |

---

### 1.6 Composer（编排器）

| 属性 | 描述 |

| **常见组合** | cursor composer, premiere composer, pixel composer |

## 六、问答/指南类词根

这类词根直接反映用户学习和解决问题的需求，是工具产品的巨大流量入口。

### 6.1 How to（如何）

| 属性 | 描述 |
|------|------|
| **常见组合** | How to get..., How to make, How to use, How to change, How to find, How to remove |
| **用户意图** | 学习新知识、掌握新技能、解决实际问题 |
| **商业机会** | 每一个 How to 长尾词背后都有工具站机会，如 "how to remove people from pictures" 可以对应图片背景移除工具 |

> 💡 **提示**：How to 类搜索是挖掘工具需求的金矿。找到高搜索量的 How to 问题，然后构建一个能直接解决该问题的工具，是非常有效的产品策略。

## 七、词根选择建议

选择正确的词根对于产品命名和 SEO 至关重要：

| 词根 | 适用场景 | 示例产品 |
|------|----------|----------|
| **Generator** | 自动化、批量生成 | AI 文案生成器 |
| **Creator** | 创意类、个性化输出 | 视频创作工具 |
| **Maker** | 简单、快速制作 | Logo 制作器 |
| **Builder** | 复杂、模块化构建 | 网站构建器 |
| **Helper** | 轻量级辅助 | 数学解题帮手 |
| **Assistant** | 深度协作 | 写作助手 |
| **Agent** | 自主执行任务 | AI 售后代理 |
| **Advisor** | 专业建议 | 理财顾问 |

## 八、实战应用

### 8.1 SEO 关键词研究

在进行关键词研究时，可以将这些词根与你的垂直领域组合：

```
[垂直领域] + [词根]
例如：
- resume builder
- code generator
- writing assistant
- recipe maker
```

### 8.2 产品命名

选择一个能准确传达产品价值的词根：

- 如果产品**自动生成**内容 → Generator
- 如果产品**辅助创作** → Creator / Maker
- 如果产品**提供帮助** → Helper / Assistant
- 如果产品**自主执行** → Agent
- 如果产品**提供建议** → Advisor

3. **发现商业机会** - 在特定词根领域找到未被满足的需求
4. **理解竞争格局** - 分析竞品使用的词根和定位
