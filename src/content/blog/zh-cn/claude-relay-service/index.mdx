## 二、搭建中转站

### 2.1 订阅

推荐在 **iOS 端订阅**，因为网页端或者桌面端订阅都需要绑定手机号和信用卡。

**App Store 礼品卡购买方式：**

1. 打开支付宝，切换地区为美国城市
2. 在搜索框输入"惠出境"并搜索
3. 点击"支付宝惠出境"
4. 选择"精品大牌折扣礼品卡"
5. 选择"App Store & iTunes USA"
6. 输入金额支付即可

> 💡 一般 3 分钟就会收到兑换码

---

### 2.2 搭建中转服务

搭建前，先准备一台海外的云服务器，我这里使用的是黑五购买的一台 **2C2G** 的 VPS。

使用的中转服务是 **Claude Relay Service**，GitHub 地址如下：

🔗 [https://github.com/Wei-Shaw/claude-relay-service](https://github.com/Wei-Shaw/claude-relay-service)

**搭建方式**：选择 `docker-compose`

**配置说明**：参数一般默认即可，我只修改了用户名和密码。

服务默认开放 **3000 端口**，输入 `http://ip:3000` 即可访问。

---

### 2.3 域名代理

考虑到安全问题，我还是用域名代理下。

虽然中转服务推荐 **Caddy**，但是我看到 CloudFlare 的证书最长 **15 年**，所以还是用了 CloudFlare 的证书。

我使用的是二级域名，如 `claude.your_site.com`。

**配置步骤：**

1. 在 CloudFlare 申请 Origin 证书
2. 配置 Nginx 代理

```nginx
server {
    listen 443 ssl;
    server_name claude.your_site.com;

    ssl_certificate /path/to/cloudflare.pem;
    ssl_certificate_key /path/to/cloudflare.key;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

---

### 2.4 访问中转服务

输入你的服务地址，如 `http://ip:3000` 或 `https://claude.your_site.com`。

![Claude Relay Service 使用统计页](/images/blog/claude-relay-service/relay-stats.webp)

输入预设的用户名和密码，登录后可以设置 Claude 账号及 API Key。

具体可看 [Claude Relay Service 官方教程](https://github.com/Wei-Shaw/claude-relay-service)，我这里配置好之后浅浅使用了下，可以看到使用的具体情况。
